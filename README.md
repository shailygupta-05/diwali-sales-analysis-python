# Diwali Sales Data Analysis using Python

## Project Overview

This project analyzes Diwali sales data to understand customer purchasing behavior, product performance, and sales trends during the festive season. The analysis was performed using Python and various data analysis libraries.

---

## Tools & Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## Project Workflow

1. Data Loading
2. Data Understanding
3. Data Cleaning
4. Handling Missing Values
5. Removing Duplicate Records
6. Exploratory Data Analysis (EDA)
7. Data Visualization
8. Business Insights & Conclusion

---

## Data Cleaning Performed

* Removed null values from the Amount column
* Removed columns containing 100% null values
* Removed duplicate records
* Verified data types for analysis

---

## Exploratory Data Analysis

The following analyses were performed:

* Gender-wise sales analysis
* Age group analysis
* State-wise sales analysis
* Product category analysis
* Customer purchasing behavior analysis
* Top customers analysis

---

## Key Insights

* Female customers contributed higher overall sales compared to male customers.
* Customers aged between 26–35 years were the most active buyers.
* Uttar Pradesh, Maharashtra, and Karnataka generated the highest sales revenue.
* Certain product categories showed higher customer demand during the festive season.
